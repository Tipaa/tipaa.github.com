Visual DASM .NET - An Integrated Development Environment for the DCPU-16 specification

Visual DASM .NET is a suite of tools designed for developing programs for the DCPU-16. It has all that a conventional IDE would be expected to contain - syntax and error highlighting, an assembler, a debugger/emulator and a disassembler.

It is under active development, and is constantly evolving. Written in C#, it should be portable to both Windows with .NET and Linux under Mono.
